# Achievable

Achievable is a goal tracking application that helps you track your progress towards achieving your goals. With built-in journaling and due dates for multi-stepped goals, Achievable helps you stay on track and motivated.

## Features

- **Goal tracking:** Set and track your goals, and monitor your progress as you work towards achieving them.
- **Journaling:** Keep a record of your progress and reflect on your achievements along the way.
- **Due dates:** Set due dates for multi-stepped goals to help you stay on track and motivated.
- **Symfony and MySQL:** Achievable is built using the Symfony PHP framework and MySQL database solution, providing a secure and robust platform for your goal tracking needs.
- **React and TailwindCSS:** The frontend of Achievable is built using React and TailwindCSS, providing a modern and responsive user interface.
